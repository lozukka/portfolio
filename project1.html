Classic WordPress -theme

Skills: PHP, CSS, JavaScript

What?

I coded a very basic classic WordPress theme. First, I took an introductory course and followed code-along tutorials on how to build a classic WordPress theme. After that, I designed my own theme. I used the same concept for the site: a fictional environmental club. In this theme, I created layouts, page structures, and used the loop function to display content on the page. The code can be found on my GitHub page.

Why?

I wanted to learn something new about WordPress. The basic course we had at school was very simple and beginner-friendly. When I found the theme development course, I decided to give it a try.
